# Portfolio CristianB-Dev

Portfolio personal que muestra mis habilidades y proyectos como desarrollador web.

## 🚀 Características

- Diseño moderno y responsivo
- Animaciones suaves
- Secciones para:
  - Lenguajes de programación
  - Habilidades personales
  - Proyectos
  - Contacto

## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome para iconos

## 📦 Estructura del Proyecto

```
portfolio/
│
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
├── images/

        ├── image-project2.avif
        └── image-project3.webp
```

## 🖥️ Instalación y Uso

1. Clona este repositorio:
```bash
git clone https://github.com/tu-usuario/portfolio.git
```

2. Abre el archivo `index.html` en tu navegador
